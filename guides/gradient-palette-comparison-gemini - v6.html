Complete Website Color System

A complete color system including pastel gradients, computed accent colors, a neutral grayscale, and functional UI colors.

Color Mixing & Accent Formula

Each pastel color is calculated using: New Color = Original × (1 - mixRatio) + White × mixRatio

Complementary accent colors are calculated by rotating the hue of the original (0% mix) colors by 180° on the color wheel and adjusting the lightness for strong contrast.

Summary

  • Pastel Palettes: Generated by mixing the original vibrant colors with different percentages of white. Higher percentages result in softer, more subtle tones.
  • Complementary Accents: A single, high-contrast palette computed from the original colors. This palette is designed to work effectively with any of the pastel variations for elements like buttons, links, and highlights.
  • Neutral Grays: A pure grayscale palette that provides a clean foundation for text, backgrounds, and borders without clashing with brand colors.
  • Functional Colors: Specific colors assigned to communicate status to the user (e.g., success, error, info), ensuring consistent user feedback.
